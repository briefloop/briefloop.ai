Scenarios

Built for buyers who bear consequences

Earnings follow-ups, competitor weeklies, compliance review, executive briefs — AI speeds up the drafting; the provenance still has to survive questioning.

Investor Relations (IR)

Pain: quarterly and disclosure data has tangled origins; answering analyst or reviewer follow-ups means hunting for the original evidence.

BriefLoop: an auto-generated Source Map with traceable coordinates lets disclosure data snap back to the original page and context in seconds.

Strategy & Investment Research

Pain: weekly competitor tracking changes constantly; multi-round editing lets unverified numbers slip in.

BriefLoop: core claims are frozen before polish, so revision rounds never mutate the facts.

Compliance & Review

Pain: manually re-verifying the sources behind AI drafts is expensive; stale data and missing evidence are hard to catch.

BriefLoop: stale or unsupported statements are flagged and blocked automatically — reviewers inspect exceptions instead of re-reading everything.

Executive Briefings

Pain: multi-source briefs drift; AI conclusions sound absolute without attribution to back them.

BriefLoop: conclusions must be backed by registered sources; the final call always belongs to a human, and that approval is recorded.

The Loop

The full journey of a claim

From source to human sign-off, every claim leaves a trail on the deterministic control plane. Claims the gate refuses go back into the repair loop — they do not sneak into the delivery bundle.

ledger-backed claim → delivery gate-blocked → repair loop solid nodes = deterministic control, dashed = AI drafting
Quickstart

Run your first brief in 15 minutes

The CLI handles workspaces, validation, and gates; the writing is done by the agent runtime you choose. The CLI never calls an LLM to write the report.

Start here Choose a path that works today
01pipx install briefloop
02briefloop new industry-weekly ./my-weekly
03briefloop run --workspace ./my-weekly --runtime operator
01git clone https://github.com/Stahl-G/briefloop.git
02cd briefloop && bash scripts/setup.sh
03source .venv/bin/activate
04briefloop claude install --repo-workdir .

Creates a workspace and operator handoff; the CLI does not call an LLM to write the brief.

Then open Claude Code and enter /briefloop new for conversational setup.

Artifacts

You deliver more than a document — you deliver the books

Every run keeps a full set of inspectable process ledgers: the claim ledger, source appendix, audit bundle, and the final delivery bundle. Click to expand examples.

Claim Ledger claim_ledger.json

A structured record of material facts, numbers, and registered sources, frozen as the factual basis for downstream writing.

Source Appendix source_appendix

A list of registered sources cited by the brief, with page and span traces when evidence spans are available.

Audit Bundle audit bundle

Quality-gate results, runtime traces, audit findings, and pre-delivery checks.

⌄
